#  Twistpad
My submission for the Hack Club Hackpad project.


This repository contains all the files needed to produce a Twistpad, created by me. The Twistpad consists of a 3x3 grid of standard keys, which are bound to F13 - F21. On the right side of the grid, there are two dials, and the top one is bound to volume while the bottom one is bound to video controls (forward, backward, etc.). I created this project over the course of nearly two weeks, using KiCad for the PCB, Fusion360 for the case design, and VSCode and Python for the firmware. 

So without further adieu, I present... the Twistpad!
